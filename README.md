# Simple Interest Calculator

This project is a simple Bash script that calculates Simple Interest based on user input.

## Purpose
The purpose of this project is to demonstrate the use of shell scripting and GitHub version control by building a simple interest calculator.

## Formula Used
Simple Interest is calculated using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## How to Use
1. Clone the repository:
   git clone https://github.com/Infinityblucy/github-final-project.git

2. Navigate into the project directory:
   cd github-final-project

3. Give execute permission:
   chmod +x simple-interest.sh

4. Run the script:
   ./simple-interest.sh

5. Enter the required values when prompted:
   - Principal amount
   - Rate of interest
   - Time period

## Example
Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
